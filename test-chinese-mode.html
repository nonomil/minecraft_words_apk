🇨🇳 汉语学习模式测试

测试说明

本页面用于测试汉语学习模式的功能是否正常。请按照以下步骤操作：

  1. 打开游戏主页面 Game.html
  2. 在档案选择界面，查看是否有"选择学习模式"选项
  3. 点击"🇨🇳 汉字学习"按钮
  4. 选择或创建一个档案，开始游戏
  5. 观察游戏中的单词显示

✅ 功能检查清单

1. UI 入口检查
  • 档案选择界面是否显示"🌍 选择学习模式"？
  • 是否有"🇬🇧 英语学习"和"🇨🇳 汉字学习"两个按钮？
  • 按钮下方是否显示"当前：xxx学习"？
2. 汉字模式显示检查
  • 黄色圆圈上是否显示中文（如"微笑"而不是"smile"）？
  • 单词卡片是否显示中文+拼音？
  • 单词测验是否使用中文？
  • 单词闸门是否使用中文？
  • 复活连线是否使用中文？
3. 设置面板检查
  • 点击"⚙️ 设置"，是否能看到"学习语言模式"下拉框？
  • 切换语言模式后，是否立即生效？
  • "显示拼音"选项是否可以切换？
  • 保存设置后，刷新页面是否保持选择？
4. TTS 语音检查
  • 汉字模式下，是否使用中文语音朗读？
  • 英语模式下，是否使用英文语音朗读？
5. 词汇数据检查
  • 是否能正常加载汉字词库？
  • 词汇是否包含拼音标注？
  • 切换模式后，词汇是否正确切换？
