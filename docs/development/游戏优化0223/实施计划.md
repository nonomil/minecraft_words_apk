## 第2项：金盔甲耐久不减少 + 被击中扣耐久

**文件：**
- `apk/src/modules/13-game-loop.js` 行 546-553 `ARMOR_DURATION_MINUTES`（配置）
- `apk/src/modules/13-game-loop.js` 行 571-604 `tickArmorDurabilityByTime()`（时间递减）
- `apk/src/modules/13-game-loop.js` 行 606-665 `damagePlayer()`（受击处理）

**⚠️ 修正：** 配置在 `13-game-loop.js`，不是 `01-config.js`。

**现状分析：**
- 时间递减：`tickArmorDurabilityByTime()` 每帧调用，基于 `Date.now()` 差值计算 loss
- 受击：`damagePlayer()` 只做减伤计算，不扣耐久
- 可能 bug：`armorEquippedAt`/`armorLastDurabilityTick` 在装备时未正确初始化

**改动：**
1. 调试 `tickArmorDurabilityByTime()` — 在 `equipArmor()` 中确保 `armorEquippedAt = Date.now()` 和 `armorLastDurabilityTick = Date.now()` 被正确设置
2. 在 `damagePlayer()` 中增加**所有护甲**被击中时扣耐久（不仅限金盔甲）：
   - 每次受击扣 `hitDurabilityCost` 百分比，按护甲等级区分：
   - leather: 8%, chainmail: 6%, iron: 5%, gold: 5%, diamond: 3%, netherite: 2%
   - 扣完后检查是否 ≤0，触发 `breakEquippedArmor()`
3. 金盔甲时长从 5 分钟缩短为 3 分钟，使时间递减更明显

**验收用例：**
1. 装备金盔甲，站立不动，观察耐久 UI 每秒递减（3 分钟内从 100% → 0%）
2. 装备金盔甲，被敌人击中一次，耐久立即减少约 5%
3. 耐久降到 0% 时显示 "金护甲 已失效" 提示并自动卸下

**改动：**
- 扩展 `digGroundBlock()` 目标检测：

**验收用例：**
1. 面对悬浮草方块，按攻击键切换到铁镐，敲 3 次后方块消失
2. 面对地面方块，行为不变（优先挖地面）
3. 悬浮方块上方如有敌人/物品，挖掉后应正常掉落

## 第5项：无防晒霜时地狱和沙漠每分钟掉1格血

**文件：** `apk/src/modules/06-biome.js` 行 593-626 `updateExtremeHeatEnvironment()`

**现状：**
- 地狱/火山：每 60 秒调用 `damagePlayer(0.5)`
- 沙漠：无环境伤害
- `damagePlayer()` 内部会经过护甲减伤和难度缩放（`diff.enemyDamageMult`、`damageUnit`）

**⚠️ 关键问题：** 环境伤害走 `damagePlayer()` 会被护甲减伤/难度缩放影响，导致实际扣血不是固定 1 格。

**改动：**
1. 新增 `applyEnvironmentalDamage(hpLoss)` 函数，**直接扣 playerHp**，绕过 `damagePlayer()` 的护甲/难度计算：
   ```javascript
   function applyEnvironmentalDamage(hpLoss) {
       playerHp = Math.max(0, playerHp - hpLoss);
       updateHpUI();
       if (playerHp <= 0) triggerGameOver();
   }
   ```
2. 修改 `updateExtremeHeatEnvironment()`：
   - 沙漠加入环境伤害判定（当前只有 nether/volcano）
   - 无防晒霜时：每 60 秒调用 `applyEnvironmentalDamage(1)` — 固定扣 1 格
   - 有防晒霜时：完全跳过（return）
3. 增加提示浮字：
   - 沙漠："☀️ 太热了！去村庄买防晒霜"
   - 地狱："🔥 地狱灼烧！需要防晒霜"

**验收用例：**
1. 无防晒霜进入沙漠，每 60 秒固定掉 1 格血（不受护甲/难度影响）
2. 无防晒霜进入地狱，每 60 秒固定掉 1 格血
3. 穿金盔甲但无防晒霜，仍然每 60 秒掉 1 格血（环境伤害忽略护甲）
4. 有防晒霜时，沙漠和地狱均不掉血
