# v1.23.0 Mini Program Migration Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 在不破坏 APK/Web 的情况下完成小程序语音迁移，形成统一 TTS Provider 架构。

**Architecture:** 新增 `tts/` 目录作为跨端语音抽象层。业务层仅调用 `speakByProvider`。平台差异收敛为 `apk/web/mini` 三个 provider；构建阶段按平台注入 provider。

**Tech Stack:** Vanilla JS, Provider Pattern, WeChat Mini Program TTS API (or uni-app wrapper), Playwright + platform mock tests

---

    if (typeof wx !== "undefined" && wx.createInnerAudioContext) {
