```javascript
// 03-audio.js
function diagnoseTts() {
  const hasSpeech = "speechSynthesis" in window && typeof SpeechSynthesisUtterance !== "undefined";
  return {
    hasSpeech,
    audioUnlocked: !!audioUnlocked,
    speechEnabled: !!settings?.speechEnabled,
    providerHint: hasSpeech ? "webSpeech" : "none"
  };
}
window.diagnoseTts = diagnoseTts;
```

**Step 4: Run test to verify it passes**

Run: `npm run test:e2e -- tests/e2e/specs/tts-v1220-diagnose.spec.mjs`  
Expected: PASS。

**Step 5: Commit**

```bash
git commit -m "feat: add web tts diagnose api"
```
