# v1.20.0 Learning Experience Upgrade Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 在不引入后端的前提下，补齐学习反馈闭环：学习统计可视化增强、成就维度扩展、答题即时音效反馈。  

**Architecture:** 复用已有档案弹窗与学习挑战模块，新增轻量统计视图组件和成就定义，不改变存档结构主干；音效通过 WebAudio 小样本生成，挂到挑战结果事件上。  

**Tech Stack:** Vanilla JS, CSS, Playwright E2E, localStorage
