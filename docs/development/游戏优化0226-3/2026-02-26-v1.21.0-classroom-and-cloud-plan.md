# v1.21.0 Classroom And Cloud Capability Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 实现班级部署基础能力：云存档、房间码排行榜、二维码词库分发。  

**Architecture:** 引入可替换后端适配层（先做 Supabase Provider），保持本地模式兼容；排行榜和房间采用 `room_code` 轻模型；词库二维码先做“文本词库编码/解码”，再叠加扫码接入。  

**Tech Stack:** Vanilla JS, Supabase JS SDK, Playwright E2E, localStorage fallback

```javascript
// 23-vocab-share-qr.js
export function encodeVocabSharePayload(text) {
  return btoa(encodeURIComponent(String(text || "")));
}
export function decodeVocabSharePayload(payload) {
  return decodeURIComponent(atob(String(payload || "").trim()));
}
window.encodeVocabSharePayload = encodeVocabSharePayload;
window.decodeVocabSharePayload = decodeVocabSharePayload;
```
