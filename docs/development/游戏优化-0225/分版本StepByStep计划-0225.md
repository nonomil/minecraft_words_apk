## 每个版本的交付标准（通用）

每个版本完成前必须满足：

1. **版本号更新**：`android-app/package.json` 的 `version` 字段更新为目标版本号
2. **CHANGELOG 更新**：在项目根目录 `CHANGELOG.md` 追加本版本条目
3. **回归测试通过**：各版本计划中列出的回归测试项全部通过
4. **git tag 推送**：`git tag vX.Y.Z && git push origin <branch> --tags`
5. **GitHub Release 创建**：`gh release create vX.Y.Z` 含版本说明
