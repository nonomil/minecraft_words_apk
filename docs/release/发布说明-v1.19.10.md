### 消耗性装备系统 (Consumable Equipment System)
- 新增：主界面新增消耗品槽位，支持投掷物、末影珍珠等背景素材装备。
- 新增：长按攻击键（800ms）触发消耗品，带圆形进度条动画反馈。
- 新增：Debuff 系统支持燃烧、减速效果，带粒子特效。

### 性能优化
- 粒子池优化：采用对象池化处理火花粒子，降低频繁 GC 带来的性能波动。

### UI 增强
- HUD 联动：实时显示消耗品图标、名称与数量，带呼吸灯特效。
- 交互优化：攻击键采用 Pointer Events API 适配多端长按逻辑。
