### 末影龙专属末地战场
- **专属战场**：新增独立 `endDragonArena`，末影龙不再只是召唤/骑乘相关功能，而是拥有完整的专属 Boss 战场。
- **正式入口**：玩家切入 `end` 群系后，会自动进入末影龙战场，而不再只依赖 debug 入口。

### 战斗与机制增强
- **三阶段战斗**：末影龙支持 phase 1 / 2 / 3，具备稳定的阶段意图与行为切换。
- **末地水晶机制**：水晶会为末影龙回血，击碎后可中断治疗并推进战斗节奏。
- **危险区与胜利出口**：加入龙息危险区、胜利状态与返回门就绪状态。

### 调试与回归验证
- **调试接口**：补充龙战场进入、退出、切阶段、伤害、水晶破坏、强制胜利等接口。
- **自动化测试**：新增龙战场 Playwright 用例与 helper，覆盖进入、阶段、水晶、危险区、胜利、群系切换与恢复。
