## v1.19.10
- 日期：2026-03-06
- 新增：消耗性装备系统（Consumable Equipment System）。
  - 专用消耗品槽位：支持投掷物、末影珍珠等背景素材装备。
  - 长按触发机制：短按普通攻击，长按（800ms）触发消耗品使用，带圆形进度条动画。
  - 减伤与 Debuff 系统：支持燃烧、减速等状态效果，带粒子特效。
  - 性能优化：粒子池对象池化，减少 GC 压力。
- UI 增强：HUD 实时显示消耗品图标、名称与数量，带呼吸灯特效。
- 发布：版本升级到 `1.19.10`（versionCode/buildNumber=65）。
