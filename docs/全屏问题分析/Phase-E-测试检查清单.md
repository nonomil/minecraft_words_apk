## E2：设备矩阵验证

### 2.1 横屏手机（短边 < 768px）
**测试设备示例**：5.5-6.5 寸手机，分辨率 1920x1080 / 2340x1080

- [ ] 启动后识别为移动布局（不误判为桌面）
- [ ] 首页三列网格不挤压，卡片可点击
- [ ] 学习界面字体可读，按钮可点击（最小 44px）
- [ ] 刘海/挖孔屏内容不被遮挡

### 2.2 小平板（短边 ≥ 768px，< 1024px）
**测试设备示例**：8-9 寸平板

- [ ] 启动后识别为移动布局或桌面布局（取决于短边阈值）
- [ ] 如果是移动布局，首页网格宽松，字体适中
- [ ] 学习界面布局不过度拉伸，内容居中

### 2.3 大平板（短边 ≥ 1024px）
**测试设备示例**：10 寸以上平板

- [ ] 启动后识别为桌面布局（如果 auto 模式）
- [ ] 如果强制移动布局，首页和学习界面布局稳定
- [ ] 字体和控件尺寸合理
