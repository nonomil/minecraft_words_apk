# Release Notes - v1.19.15

**Release Date**: 2026-03-06

## 🎯 Language-Specific Learning Challenges

This release introduces differentiated challenge types for Chinese and English learning modes, plus voice assistance features to improve the learning experience.

### ✨ New Features

#### Chinese Mode Challenges
- **Character-Based Fill Blank**: In Chinese mode, challenges now ask "缺少哪个汉字？" (Which Chinese character is missing?) instead of "缺少哪个字母？" (Which letter is missing?)
  - Displays Chinese characters with one missing: 学 _ 中文 → 学 习 中文
  - Options show Chinese characters instead of English letters
  - Automatically generates similar character distractors from vocabulary

- **Character Scrambling**: Chinese mode uses character-based unscrambling instead of letter-based
  - Shows scrambled Chinese characters: 习 中 学 文
  - Prompt: "重新排列汉字，拼出正确词组" (Rearrange characters to form correct phrase)
  - Options show complete Chinese phrases

#### Voice Assistance
- **Word Gate Voice Prompts**: When player encounters a word gate, the word/phrase is automatically spoken
  - Helps players understand what they need to answer
  - Respects language mode (Chinese mode speaks Chinese, English mode speaks English)

- **Challenge Option Voice Hints**: Hover mouse or focus on any challenge option to hear it spoken
  - Especially helpful for Chinese character recognition
  - Works on both desktop (hover) and mobile (focus)
  - Respects speech settings and language mode

### 🔧 Technical Implementation

**Challenge Generation Logic**:
```javascript
// Checks language mode and routes to appropriate challenge generator
function generateFillBlankChallenge(wordObj) {
    const languageMode = getLanguageModeSafe();
    if (languageMode === "chinese") {
        return generateChineseFillBlankChallenge(wordObj);
    }
    // English mode logic...
}
```

**New Functions**:
- `generateChineseFillBlankChallenge()` - Chinese character fill_blank
- `generateChinesePhraseFillBlankChallenge()` - Multi-character phrases
- `generateChineseUnscrambleChallenge()` - Character scrambling
- `generateChineseCharOptions()` - Similar character generation
- `generateChineseScrambleDistractors()` - Fake scrambled phrases

### 📊 Challenge Type Matrix

| Challenge Type | English Mode | Chinese Mode |
|---------------|--------------|--------------|
| Fill Blank | Missing letter (a b _ d) | Missing character (学 _ 中文) |
| Unscramble | Scrambled letters (t a c) | Scrambled characters (习 学 中) |
| Translate | English → Chinese | Chinese → English |
| Listen | Hear English word | Hear Chinese word |
