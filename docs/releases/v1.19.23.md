# Release Notes - v1.19.23

**Release Date**: 2026-03-08

## 🏘️ Village Interaction Fixes

This release resolves critical regressions in village interaction systems, improving reliability and user experience for word gates, trader interactions, and word house quizzes.

### 🐛 Bug Fixes

#### Village Interaction Regressions
- **Word Gate Toggle**: Fixed runtime behavior for word gate enable/disable
  - Gates now properly respect `settings.wordGateEnabled` during both trigger and render
  - Existing spawned gates become inert and invisible when disabled
  - No more unexpected gate triggers after toggling settings

- **Trader Auto-Enter**: Corrected trader interaction radius and entry behavior
  - Increased interior action range for more reliable interaction
  - Fixed auto-enter detection from larger distances
  - Improved sell grid UI with multi-column layout

- **Word House Trigger**: Enhanced word house quiz flow reliability
  - Made word house quiz mandatory (checkpoint mode)
  - Removed cancel/exit affordances during forced quiz mode
  - Players must complete all questions before leaving

- **Village Challenge Quiz**: Resolved state conflicts in quiz flow
  - Fixed quiz modal state management
  - Improved interaction between village challenges and word house system
