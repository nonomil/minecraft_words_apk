# Release Notes - v1.19.21

**Release Date**: 2026-03-07

## Ender Dragon End Arena

This release upgrades the dragon feature set from summon/riding mechanics into a full dedicated End battle encounter.

### Highlights

- **Dedicated End Arena**
  - Added a separate `endDragonArena` lifecycle instead of overloading the ground-boss system.
  - Entering the `end` biome now transitions into the Ender Dragon battle flow.
- **Multi-Phase Dragon Boss**
  - Added stable phase behaviors for phase 1 / 2 / 3.
  - Added intent keys for orbit healing, dive pressure, breath attacks, and low-altitude frenzy.
- **Crystal and Hazard Mechanics**
  - End crystals can heal the dragon during the early phase.
  - Crystal destruction interrupts healing and reduces arena pressure.
  - Breath hazards and portal-ready victory state are now part of the encounter loop.
- **Debug and Automated Verification**
  - Added dragon arena debug controls and E2E helpers.
  - Added targeted Playwright coverage for arena entry, phase changes, damage, crystals, hazards, victory, biome switching, and restoration.
