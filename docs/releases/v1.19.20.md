This release completes the implementation of the EnderDragon summoning and riding mechanics, bringing a major gameplay feature to the game.

- **EnderDragon Entity**: Complete AI behavior with fireball attacks

  - Ground fire mechanics integration
- **Debug Tools**: Manual debug coverage for dragon summon functionality

  - New E2E test suite for dragon and gunpowder mechanics
