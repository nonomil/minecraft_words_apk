# Release Notes - v1.19.13

**Release Date**: 2026-03-06

## 🌍 Bilingual Learning Mode

This release introduces a complete dual-language learning system, allowing users to choose between English vocabulary learning and Chinese character learning.

### ✨ New Features

#### Language Mode Selection
- **First-Launch Onboarding**: New users are greeted with a language selection modal
  - 🇬🇧 English Learning Mode
  - 🇨🇳 Chinese Character Learning Mode
- **Settings Integration**: Language mode can be changed anytime in Settings panel
- **Pinyin Display Toggle**: Optional pinyin display for Chinese characters

#### Chinese Vocabulary Support
- **Kindergarten Chinese Characters Pack** (幼儿园汉字)
  - Curated vocabulary for early learners
  - Integrated with existing game mechanics
  - Full support for word cards, challenges, and TTS

#### Smart TTS Language Switching
- **Chinese Mode**: Only speaks Chinese pronunciation
- **English Mode**: Speaks English first, then Chinese (if enabled)
- **Language-Aware Settings**: Separate speech rate controls for each language

#### Data Migration System
- Automatic migration to bilingual mode (v2.2.0)
- Separate progress tracking for English and Chinese modes
- Backward compatible with existing saves

### 🔧 Technical Improvements

#### Vocabulary System
- **Browser-Compatible Manifest**: Converted vocabulary manifest to window global format
- **Bilingual Vocabulary Functions**:
  - `normalizeWordContent()` - Normalize word entries
  - `getCurrentLanguageMode()` - Get active language mode
  - `filterWordsByLanguageMode()` - Filter vocabulary by mode
  - `getDisplayContent()` - Transform display based on language
