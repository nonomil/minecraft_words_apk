## **一、产品综合评估**

### **1.1 优势分析（真实存在的差异化）**

这款游戏的核心竞争力在于强制学习机制——不答单词就过不了关，而不是游戏的附加模块。这一点让它和市面上99%的教育类游戏区分开来。

| 优势项 | 说明 |
| :---- | :---- |
| **强制学习闭环** | 单词门、复活测验、填空挑战无缝嵌入战斗流程，孩子想继续玩必须作答 |
| **双IP吸引力** | Minecraft视觉 × 马里奥操作感，命中6-14岁男孩核心喜好 |
| **词库分级清晰** | 幼儿园→初中五个阶段，家长看得懂、选得对 |
| **无广告无内购陷阱** | 父母最反感孩子被广告引导，这是最强情感卖点 |
| **离线可用** | 不依赖网络，随时随地学，出门旅行也能用 |
| **TTS双语朗读** | 英中顺序朗读，有真实听力辅助效果，不是摆设 |
| **持续迭代** | 57个Release版本，说明开发者在认真打磨产品 |

### **1.2 当前核心问题**

## **二、版权风险处理（最优先，必须先做）**

| ⚠️ 风险说明 |
| :---- |
| 当前游戏使用了「马里奥」和「我的世界/Minecraft」两个受保护IP的名称与视觉元素。 任天堂和微软均有积极的版权执法记录。公开商业化销售可能触发：DMCA投诉、应用商店下架、律师函。 私下小规模传播风险较低，但一旦做大或上应用商店，风险显著上升。 |

### **2.1 三种处理方案（按推荐度排序）**

| 方案 | 说明 |
| :---- | :---- |
| **方案A（推荐）换皮+改名** | 保留玩法，把Minecraft方块风格换成原创像素风角色（如自创的「方块小队」），名称改为「像素词语冒险」或「方块英语大冒险」。视觉改动成本中等，规避所有IP风险，可正式上架应用商店。 |
| **方案B（短期）内部/私域传播** | 不对外公开商业销售，仅在小范围（家长群、学习群）以「工具共享」形式收取象征性费用或运营费。风险低但规模有限，适合先验证付费意愿。 |
| **方案C（风险高）直接商业销售** | 保留现有名称和元素，直接对外销售。短期可行，但一旦规模化或引起关注，面临较高法律风险，不推荐。 |

**建议路线：先走方案B验证市场（1-2个月），确认有付费需求后投入资源做方案A的换皮，然后正式上架。**

## **三、目标用户与购买决策分析**

### **3.1 主力用户画像**

购买决策者是父母，使用者是孩子。需要同时打动两类人。

| 类型 | 描述 | 关注点 |
| :---- | :---- | :---- |
| 核心用户 | 6-12岁男孩的家长（尤其是孩子已经是Minecraft粉丝的） | 购买决策者，关注安全/效果/价格 |
| 次要用户 | 孩子本身（会主动要求家长购买） | 游戏体验驱动，关注好不好玩 |
| 潜在B端 | 家庭教师、英语培训班、小学英语老师 | 可批量授权，单价更高 |

### **3.2 家长会买单的核心条件**

* 孩子已经在玩Minecraft——不需要解释游戏，直接卖「边玩边学」的概念

* 看到真实效果证明——最好是「孩子玩了一周多背了XXX个单词」的截图或故事

* 价格在可接受范围——20-50元一次性买断，心理门槛比订阅低

* 无广告承诺——家长最怕孩子游戏中被乱七八糟的广告引导

* 有试用机会——免费体验前3关，降低决策风险

### **3.3 父母的顾虑与应对话术**

| 家长顾虑 | 建议应对方式 |
| :---- | :---- |
| **顾虑：孩子只顾打游戏不学单词** | 应对：单词门机制不填对就过不了关，不是可选的，是强制的 |
| **顾虑：游戏会让孩子上瘾** | 应对：可设置每日时长限制（如30分钟），家长管控方便 |
| **顾虑：不知道学了什么、学了多少** | 应对：游戏内词汇统计面板，家长随时查看已学单词 |
| **顾虑：安装未知APK不安全** | 应对：提供应用商店版本（上架后）；短期提供病毒扫描截图背书 |
| **顾虑：和学校课程不对应** | 应对：明确标注词库对应人教版/外研版哪些年级单元 |

## **七、产品改进建议（提升商业价值）**

### **7.1 高优先级（直接影响购买转化）**

| 功能 | 原因 |
| :---- | :---- |
| **家长仪表盘** | 增加「我家孩子本周学了N个单词、正确率X%」的数据报表页面，家长最爱看 |
| **课程进度感** | 增加「今日任务：完成3关+学10个新词」的每日目标系统，增加成就感 |
| **分享截图功能** | 游戏内一键生成「孩子今日学习成绩单」分享图，家长主动传播即是最好的广告 |
| **词库对应课本单元** | 在词库选择界面注明「对应人教版三年级上册Unit 1-3」，家长选词库更有目标 |
| **试玩前三关免费** | 设置前3关免费，降低首次尝试门槛，体验后付费转化率更高 |

### **7.2 中优先级（提升留存与口碑）**

* 成就系统：「单词猎人」「关卡王者」等称号，孩子愿意炫耀就会主动拉同学

* 双人模式：和同学/兄弟姐妹PK词汇量，增加社交属性

* 错词复习：自动收集答错的单词，隔天复习强化，真正实现科学记忆

* iOS版本：Capacitor本身支持，扩大潜在用户群约40%

* 家长时间管控：每日游戏时长上限设置（消除家长最大顾虑之一）
