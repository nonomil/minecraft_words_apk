# vB 学习体验增强计划（P1，落地版）

## 目标

把“玩得爽”升级为“学得见”，直接提升家长感知和复购概率。

## 周期

1. Day 3-4：学习统计增强。
2. Day 5：成就系统深化。
3. Day 6：答题反馈音效。

## Task 1：学习统计增强

文件：
1. `src/modules/08-account.js`
2. `src/styles.css`

步骤：
1. 新增最近7日学习趋势模块。
2. 新增本周 vs 上周对比文案。
3. 弱词区增加“专项练习（占位）”按钮。

验收：
1. 档案页能看到趋势、对比、弱词行动入口。
2. 空数据时不报错，有兜底文案。

## Task 2：成就系统深化

文件：
1. `src/modules/01-config.js`
2. `src/modules/08-account.js`

步骤：
1. 增加连续学习成就（3天/7天）。
2. 增加完美局成就（单局0错）。
3. 增加隐藏成就（夜间时段触发）。
4. 每个成就显示进度条。

验收：
1. 新成就可显示。
2. 对应行为发生时进度可增长。

## Task 3：答题即时音效

文件：
1. `src/modules/03-audio.js`
2. `src/modules/12-challenges.js`
3. `src/modules/16-events.js`
4. `Game.html`

步骤：
1. 增加答对短音效。
2. 增加答错短音效。
3. 连击时提高音调反馈。
4. 设置中增加独立音效开关。

验收：
1. 挑战答题反馈音可用。
2. 关闭开关后完全静音。
