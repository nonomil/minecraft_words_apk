# 召唤机制需求文档

> 创建时间：2026-03-06
> 状态：需求讨论中

## 需求理解

根据 `计划-0306.md`，本次需求包含两个主要功能：

### 1. 末影龙召唤系统
- **触发方式**：宝箱开出龙蛋道具
- **召唤机制**：使用龙蛋可召唤末影龙
- **骑乘功能**：主角可跳到末影龙身上，实现飞行
- **攻击能力**：末影龙可喷火球攻击敌人
- **生命值**：30点（雪傀儡的2倍）

### 2. 火药效果增强
- **当前效果**：火焰范围较小
- **目标效果**：主角前方所有地面都起火
