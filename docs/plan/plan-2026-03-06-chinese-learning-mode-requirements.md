# 需求文档：汉语学习模式

**创建时间**: 2026-03-06
**状态**: 需求讨论中

## 需求理解

用户希望添加"汉语学习模式"功能：

1. **游戏内容变化**：在汉语学习模式下，黄色圆圈上显示的单词要切换成汉字或短语（而不是英文单词）
2. **UI 入口**：在游戏刚进入的页面，第二个弹窗（选择档案之后），增加"英语学习"和"汉语学习"的选项
