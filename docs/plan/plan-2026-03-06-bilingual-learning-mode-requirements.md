# 需求文档：双语学习模式（英语+汉字）

> 创建时间：2026-03-06
> 状态：需求讨论中

## 1. 需求理解

### 核心需求
在当前游戏版本中增加"汉字学习模式"，使游戏支持：
- **英语学习模式**（现有）：学习英文单词、短语
- **汉字学习模式**（新增）：学习汉字、拼音、词组

### 用户场景
- 幼儿园小朋友需要同时学习英语和汉字
- 家长/老师希望一个 App 解决所有学习需求
- 用户可以在两种模式间切换

### 功能要求
1. **学习内容**：
   - 汉字显示
   - 拼音标注
   - 词组学习
   - 英文翻译（辅助）

2. **切换方式**：
   - 设置页面：全局切换学习模式
   - 首页弹窗：快速切换按钮

3. **兼容性**：
   - 在当前版本基础上扩展
   - 不影响现有英语学习功能
   - 进度分别记录

## 5. 风险评估

### 高风险点
1. **向后兼容性**：现有用户的存档数据不能丢失
2. **词库数据质量**：汉字词库需要准确的拼音标注
3. **UI 适配**：汉字+拼音的显示空间需求 > 纯英文

### 中风险点
1. **TTS 质量**：中文 TTS 发音准确性
2. **模式切换**：用户误操作导致进度混乱
3. **性能影响**：词库数据量增加

### 低风险点
1. **代码复杂度**：现有架构支持扩展
2. **测试覆盖**：可以复用现有测试框架
