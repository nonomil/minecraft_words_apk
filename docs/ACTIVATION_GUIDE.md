## 🎯 使用方法

### 用户激活流程

1. **打开应用**
   - Web版: 访问网址
   - APK版: 打开应用

2. **进入设置**
   - 点击"设置"按钮
   - 滚动到"激活"部分

3. **输入激活码**
   - 在"激活码"输入框中输入激活码
   - 点击"立即激活"按钮

4. **验证成功**
   - 显示"激活成功"提示
   - 解除20个单词限制
   - 可以学习所有单词

## ⚠️ 常见问题

### Q1: 激活码无效
**可能原因**:
- 输入错误(注意大小写)
- 网络问题(无法连接验证服务器)
- 激活码已过期(加密激活码)

**解决方法**:
1. 检查输入是否正确
2. 检查网络连接
3. 尝试离线验证
4. 联系客服

### Q2: 激活后仍有限制
**可能原因**:
- 使用了调试密码(不解除限制)
- 激活信息未保存

**解决方法**:
1. 确认使用的是激活码而非调试密码
2. 重新激活
3. 清除缓存后重试

### Q3: 一个激活码可以用几次?
**当前实现**: 
- 简单激活码: 可重复使用
- 加密激活码: 可重复使用

**建议**:
- 每个激活码只发放给一个用户
- 升级到服务器验证后可限制次数

### Q4: 如何批量激活?
**当前不支持**,每个设备需要单独激活。

**未来计划**:
- 企业版: 支持批量激活
- 教育版: 支持班级激活

## 🛠️ 技术说明

### 验证流程

```
用户输入激活码
    ↓
前端格式检查
    ↓
尝试在线验证 (GitHub Raw)
    ↓ (失败)
尝试本地验证 (docs/activation_codes.txt)
    ↓ (失败)
尝试加密验证 (解码并检查过期时间)
    ↓
返回验证结果
```

### 存储位置

**浏览器**: `localStorage.activationInfo`
```json
{
  "activated": true,
  "code": "MC-2024-L8K9M-3N4P",
  "debug": false,
  "ts": 1732723800000,
  "source": "https://raw.githubusercontent.com/..."
}
```

**Android**: WebView LocalStorage
- 位置: `/data/data/com.example.mcwords/app_webview/Local Storage/`
- 更新APK时数据保留
