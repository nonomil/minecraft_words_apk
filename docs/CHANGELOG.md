## [2.2.3] - 2026-02-18

### ✨ 新增
- **词库治理工具链**
  - 新增词库抽取、校验、覆盖率对比、候选清单生成脚本
  - 新增词库自动修复脚本（空中文、图片链接与 URL 规范化）
  - 新增核心去重词库构建脚本与合并报告

### 🔧 优化
- 词库质量门禁流程标准化（审计、修复、复检可重复执行）
- 词库维护流程文档与技能文档补齐，支持后续持续增删改

### 🐛 修复
- 修复词库中的空中文字段
- 修复图片 URL 空值与 Unsplash 参数异常格式
- 修复状态效果词条缺失图片的问题

## 版本类型说明

- **新增 (Added)**: 新功能
- **优化 (Changed)**: 现有功能的变更
- **弃用 (Deprecated)**: 即将移除的功能
- **移除 (Removed)**: 已移除的功能
- **修复 (Fixed)**: Bug修复
- **安全 (Security)**: 安全相关更新
