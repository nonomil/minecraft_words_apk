# 马里奥×我的世界 单词冒险（Android APK）

基于横版过关玩法，融合“我的世界”主题元素，把英语单词学习放进战斗、收集与挑战流程。玩家在不同群系中闯关，击败怪物与 BOSS，收集材料与宝箱奖励，同时完成单词学习与挑战任务。项目使用 Capacitor 打包为 Android APK，并提供自动化构建与发布流程。

## 主要功能
- 横版闯关 + 单词学习卡片 + 词库轮换
- 触摸按键与键盘操作并存，支持手机/平板显示模式
- 本地构建与 GitHub Actions 在线构建

## 功能介绍
- **关卡推进**：平台跳跃、战斗与地形互动贯穿流程
- **群系切换**：森林、樱花林、雪地、沙漠、蘑菇岛、山地、海洋、火山、下界、深暗、天空之城等多环境
- **BOSS 战**：凋零、恶魂、烈焰人、凋零骷髅等阶段战，包含冲刺、追踪弹、火焰旋风、格挡破防等机制
- **学习机制**：
  - 核心玩法：单词卡（拼写/释义/发音/例句）、填空挑战与复活测验按节奏插入
  - 内置词库：幼儿园、小学低/高年级、初中分级词汇及 Minecraft 游戏专属词汇
- **词库轮换**：多词库自动轮换，支持手动切换
- **物品与装备**：背包物品、武器与护甲状态展示
- **宝箱与奖励**：探索宝箱、收集素材与稀有掉落
- **触控体验**：左右移动、跳跃、攻击、交互与切换武器

## 📚 沉浸式英语学习系统

游戏将英语学习深度融合在核心玩法中，旨在通过游戏化的方式帮助玩家记忆单词和短语。

### 1. 丰富的词库选择
游戏内置了分级明确的词汇库，适合不同阶段的学习者：
*   **幼儿园 (Kindergarten)**: 基础认知词汇。
*   **小学低年级 (Elementary Lower)**: 日常生活与基础学科词汇。
*   **小学高年级 (Elementary Upper)**: 进阶词汇与短句。
*   **初中 (Junior High)**: 课标词汇与常用短语。
*   **我的世界 (Minecraft)**: 游戏内物品、方块与生物的英文名称。

另外，词库是以“词库包（Pack）”的形式组织的：每个阶段都提供 **初级 / 中级 / 高级 / 完整** 四种粒度，玩家可在设置面板中手动选择，也可使用“随机词库（按类别轮换）”自动轮换（`vocabSelection=auto`）。

| 阶段 | 可选词库包（示例） | 说明 |
|---|---|---|
| 幼儿园 | `vocab.kindergarten.basic / intermediate / advanced / vocab.kindergarten` | 分卷 + 合并词库（含补充词库） |
| 小学低年级 | `vocab.elementary_lower.basic / intermediate / advanced / vocab.elementary_lower` | 分卷组合；另有“小学-补充”包 |
| 小学高年级 | `vocab.elementary_upper.basic / intermediate / advanced / vocab.elementary_upper` | 以合并词库为主 |
| 初中 | `vocab.junior_high.basic / intermediate / advanced / vocab.junior_high` | 分级 + 完整包 |
| Minecraft 主题 | `vocab.minecraft.basic / intermediate / advanced / vocab.minecraft` | Blocks / Items / Entities / Biomes / Environment 等主题集合 |

### 2. 多样化的学习模式
*   **单词卡片**: 拾取单词道具时会弹出单词卡，展示 **英文拼写**、**中文释义**，并支持展示 **短语/例句**（若词条包含 `phrase/phraseZh` 等字段）。
*   **配图记忆**: 词条包含图片链接（`imageUrl` 或 `images/imageURLs`）时，单词卡会显示配图；未提供图片则不显示。
*   **英文+中文朗读**: 每次触发单词卡时，会按顺序朗读 **英文 →（可选）中文**。优先使用 APK 原生 TTS（Capacitor TextToSpeech），否则使用浏览器 Web Speech；若环境不支持，再回退到在线音频朗读。
*   **学习挑战（题型）**：在跑酷战斗流程中穿插弹窗挑战，常见题型包括：
    *   **翻译选择**：看到英文，选择正确中文释义（四选一）。
    *   **听音选词**：听发音，选择正确英文（四选一）。
    *   **单词填空**：缺 1 个字母，选择正确字母补全。
    *   **多字母填空**：一次缺 2 个字母，选择正确组合补全（较长单词更常见）。
    *   **重排拼写**：字母/单词顺序被打乱，选择正确答案拼回单词/短语。
    *   默认规则：单题限时 15 秒；基础“干扰项数”为 3（因此通常为四选一）；答对奖励 **+20 分 +1💎**，答错扣分（默认 -8 分）。
*   **单词门（Word Gate）**：部分路段会生成“单词门”，必须完成一次填空挑战才能通过，避免无脑冲关。
    *   生成规则：需要先达到一定分数后才会出现（默认分数 ≥ 500），且生成是概率事件（默认约 8%）。
*   **复活测验（学习复活）**：死亡后会优先尝试“单词连线复活”小游戏（将英文与中文匹配连线，只有 1 次机会；成功可原地复活继续）。
    *   触发条件：本局累计学习到的“不同单词”达到一定数量后才会开启（避免一上来就复活无门槛）。
    *   判定规则：倒计时内完成 5 组连线，至少匹配对 4 组才算成功；成功后恢复 3 点生命值，并额外奖励积分（每组匹配默认 +10 分）。
    *   若本局学习单词不足或复活失败，则进入“游戏结束”界面，可选择 **钻石复活** 或 **积分复活** 后继续。
        *   **钻石复活**：默认消耗 10💎，直接满血继续。
        *   **积分复活**：默认消耗 500 分，恢复 50% 生命并获得短暂无敌时间（默认约 3 秒）。

### 3. 学习奖励
*   **资源获取**: 正确回答问题可以获得钻石、箭矢等稀有资源。
*   **村庄挑战**: 在村庄的“单词学习屋”中完成挑战，可获得大量分数和特殊奖励。

## 🌍 关卡与环境（详细图鉴）

游戏世界由多个风格迥异的 **生物群系 (Biomes)** 组成，随着分数的增加，玩家会穿越不同的环境。每个群系都有独特的视觉风格、背景音乐和专属敌人。

### 群系推进规则（跑图为什么会“换地图”）
游戏使用“分数驱动的轮换”来切换群系（可在设置中调整切换速度）。整体机制包括：
- **切换步长（stepScore）**：每累计一定分数，群系轮换一次（默认 300 分/步；设置里可覆盖，但不低于 150）。
- **解锁阈值（unlockScore）**：分数达到阈值才会加入轮换池：

| 群系 | 解锁分数 |
|---|---|
| Forest | 0 |
| Cherry Grove | 100 |
| Snow | 250 |
| Desert | 400 |
| Mushroom Island | 550 |
| Mountain | 800 |
| Ocean | 1100 |
| Volcano | 1500 |
| Nether | 2100 |
| Deep Dark | 2800 |
| End | 3800 |
| Sky Dimension | 5000 |

- **最小停留（minStay）**：切换前需要同时满足“在本群系拿到的分数 + 停留时间”：
  - Forest：≥100 分 且 ≥20 秒
  - Cherry Grove：≥60 分 且 ≥10 秒
  - Snow / Desert：≥150 分 且 ≥30 秒
  - Mushroom Island：≥200 分 且 ≥40 秒
  - Mountain：≥200 分 且 ≥35 秒
  - Ocean：≥140 分 且 ≥25 秒
  - Volcano：≥320 分 且 ≥70 秒
  - Nether：≥380 分 且 ≥80 秒
  - Deep Dark：≥320 分 且 ≥60 秒
  - End / Sky Dimension：≥300 分 且 ≥50 秒

- **BOSS 门禁（Biome Gate）**：部分群系切换会先进入门禁战，胜利后才允许切换（默认 Forest 免门禁；每个群系“首次离开”最多触发 1 次；凋零门禁会“每两次切换触发一次”以控制节奏）。

### 敌人“轮次”与中毒规则（重要）
> 本节是“为什么同一张地图越跑越难 / 为什么有时不掉中毒”的规则说明。

- **轮次（round）的含义**：轮次不是时间，而是“你进入该群系的次数”。例如第一次进入海洋是 round 1，第二次回到海洋就是 round 2（会解锁更强敌人/更高精英概率）。
- **轮次敌人池（enemyTiers）**：部分群系在 `config/biomes.json` 里配置了 `enemyTiers`，每一轮会指定：
  - `types`：本轮主要敌人类型
  - `spawnRate`：本轮“刷怪触发后”是否真正生成的概率（未命中会跳过生成）
  - `eliteChance`：精英概率（更强/更稀有掉落）
  - `poison`：本轮是否允许“中毒效果”生效
- **中毒效果的全局门槛**：
  - 只有当 **分数 ≥ 600** 且当前轮次 `poison=true` 时，中毒类攻击（蜜蜂/女巫/河豚等）才会真正让玩家进入“中毒状态”
  - 中毒持续 **5 分钟**，并以“每分钟累计半格”的方式结算伤害：大约 **每 2 分钟掉 1 格血**（最多约掉 2 格血）

- **森林（Forest）**：
  - **环境**: 郁郁葱葱的橡树、白桦树，藤蔓垂下的洞穴入口；常规移动与战斗节奏。
  - **特色**: 初始区域，相对安全，适合熟悉操作。
  - **敌人池**: 以经典怪为主（僵尸/苦力怕/蜘蛛/骷髅/末影人）；低分段会限制末影人出现，避免早期难度跳变。

- **樱花林（Cherry Grove）**：
  - **环境**: 梦幻的粉色落叶，日式风格的背景；花瓣粒子效果。
  - **敌人**: **蜜蜂 (Bee)**（近战+中毒）、**狐狸 (Fox)**（偷分）、**女巫 (Witch)**（毒与远程药水威胁）。
  - **出怪节奏**: 以“轮次”推进（round 1~3 逐步加入孢子虫/狐狸等），并会穿插专属敌人逻辑（例如狐狸在樱花林更高概率出现，且同屏会限制重复刷出）。
  - **轮次敌人（配置）**:
    - round 1：Bee（spawnRate=0.5，poison=false）
    - round 2：Bee + Spore Bug（spawnRate=0.8，poison=true，eliteChance=0.1）
    - round 3：Bee + Spore Bug + Fox（spawnRate=1.0，poison=true，eliteChance=0.2）
  - **女巫细节**：进入追击距离后会投掷药水；若满足“中毒可生效”则约 60% 概率为毒药水，否则多为减速药水。

- **雪地（Snow）**：
  - **环境**: 漫天飞雪，冰刺耸立，地面覆盖厚厚的积雪。
  - **特色**: 视觉对比强，平台偏稳，但移动速度会加快（滑行感）。
  - **环境影响**: 角色移动速度倍率上调（默认 1.2）。

- **沙漠（Desert）**：
  - **环境**: 烈日炎炎，仙人掌丛生，偶尔会有沙尘暴和热浪。
  - **特色**: 移动减速；视野受限，需留意隐藏在沙丘后的危险与骨架地形。
  - **环境影响**: 角色移动速度倍率下调（默认 0.85）；沙尘暴/热浪会增强压迫感。
  - **高温 DOT**: 若没有高温防护且未穿下界合金盔甲，会触发 **每 60 秒掉 1 格血** 的持续灼烧（提示会引导去村庄购买防晒霜）。
  - **对策**: 在村庄商人处购买“防晒霜”可显著缓解沙漠高温带来的负面影响（并同时作为高温防护使用）。

- **蘑菇岛（Mushroom Island）**：
  - **环境**: 巨大的红白蘑菇，紫色的菌丝地面。
  - **敌人**: **孢子虫 (Spore Bug)**（死亡分裂机制），**蘑菇牛 (Mooshroom)**（低威胁）。
  - **出怪节奏**: 轮次推进会逐步加入女巫（远程毒药水）；“中毒”在该群系的轮次中更容易被允许生效。
  - **轮次敌人（配置）**:
    - round 1：Mooshroom（spawnRate=0.4，poison=false）
    - round 2：Mooshroom + Spore Bug（spawnRate=0.7，poison=true，eliteChance=0.1）
    - round 3：Mooshroom + Spore Bug + Witch（spawnRate=1.0，poison=true，eliteChance=0.2）

- **山地/矿洞（Mountain/Cave）**：
  - **环境**: 矿脉、晶体与熔岩池分布；黑暗与雾气强化。
  - **特色**: 复杂地形，需小心跌落与熔岩。
  - **环境影响**: 黑暗度提高（默认 darkness≈0.3），更依赖对平台节奏的把握。

- **海洋（Ocean）**：
  - **环境**: 气泡、水位变化、沉船遗迹。
  - **敌人**: 溺尸、河豚与守卫轮次出现。
  - **水域规则**: 当角色进入水位线以下时，移动速度会被显著削减（更像“游泳/下潜”手感），需要提前预判落点与敌人位置。
  - **轮次分层**: round 1→溺尸；round 2→加入河豚并允许中毒；round 3→加入守卫并提高精英概率。
  - **轮次敌人（配置）**:
    - round 1：Drowned（spawnRate=0.5，poison=false）
    - round 2：Drowned + Pufferfish（spawnRate=0.8，poison=true，eliteChance=0.1）
    - round 3：Drowned + Pufferfish + Guardian（spawnRate=1.0，poison=true，eliteChance=0.2）

- **火山（Volcano）**：
  - **环境**: 熔岩流淌，黑曜石地面，空气中飘浮着灰烬粒子与环境伤害。
  - **敌人**: **岩浆怪 (Magma Cube)**（高伤近战分裂）、**火灵 (Fire Spirit)**（高移速打击）。
  - **环境影响**: 作为高温环境，若没有高温防护（防晒霜/抗火）且未穿下界合金盔甲，会触发“高温 DOT”：**每 60 秒固定掉 1 格血**（绕过护甲/难度缩放）。
  - **轮次敌人（配置）**:
    - round 1：Magma Cube（spawnRate=0.5，poison=false）
    - round 2：Magma Cube + Blaze（spawnRate=0.8，poison=false，eliteChance=0.1）
    - round 3：Magma Cube + Blaze + Wither Skeleton（spawnRate=1.0，poison=false，eliteChance=0.25）

- **下界（Nether）**：
  - **环境**: 红色调，火焰粒子，移动速度降低。
  - **敌人**: 僵尸猪灵、烈焰人与恶魂轮次。
  - **入场惩罚**: 无高温防护时，首次进入下界会额外扣 1 点生命值（作为“热浪冲击”）。
  - **环境影响**: 角色移动速度倍率下调（默认 0.7）；灵魂沙区域会额外减速。
  - **高温 DOT**: 若没有高温防护且未穿下界合金盔甲，会触发 **每 60 秒掉 1 格血** 的持续灼烧。
  - **岩浆池**: 没有防晒霜时“碰到基本即死”；有防晒霜时会把致命伤改为每 ~0.9 秒最多扣 1 格血并把角色顶回安全位置（仍然很危险）。
  - **轮次敌人（配置）**:
    - round 1：Zombie Pigman（spawnRate=0.6）
    - round 2：Zombie Pigman + Blaze（spawnRate=0.8，eliteChance=0.15）
    - round 3：Zombie Pigman + Blaze + Ghast（spawnRate=1.0，eliteChance=0.25）

- **深暗（Deep Dark）**：
  - **环境**: 幽暗恐怖的地下城，满布幽匿块。
  - **噪音系统**: 跑动、攻击或跳跃增加“噪音值”；噪音冲击波干扰。
  - **敌人**: **幽匿蠕虫 (Sculk Worm)**（轻甲快跑）、**暗影潜行者 (Shadow Stalker)**（高伤近战）、**监守者 (Warden)**（高血高伤，噪音封顶触发）。
  - **对策**: 需装备 **静音靴** 减少噪音积累。
  - **关键细节**:
    - 噪音条范围 0~100，会随时间缓慢衰减；达到 100 且不在冷却期时会强制唤醒 Warden。
    - 静音靴会把“跳跃/攻击/用道具”的噪音增量压低到约 55%，并按动作消耗耐久（耐久耗尽会损坏）。
    - 静音靴初始耐久为 30；每次关键动作消耗 1 点；降噪后的噪音增量最低不会低于 3。
    - Warden 除了近战重击外，还会在中距离释放“音波冲击波”（可命中并造成高额伤害）。

- **天空之城（Sky Dimension）**：
  - **环境**: 漂浮在云端的岛屿，梦幻般的天空。
  - **特色**: 低重力环境，支持高跳跃。
  - **敌人**: **幻翼 (Phantom)**（俯冲攻击）、**恼鬼 (Vex)**（穿墙高机动连击）。
  - **环境影响**: 重力降低、跳跃增强（默认 gravity≈0.8，jump≈1.3），并存在风场把角色推向左/上/右；上升风场会额外生成奖励平台与宝箱，鼓励探索。

## ⚔️ BOSS 图鉴（阶段机制）

在关卡转换或特定阶段，玩家将遭遇强大的BOSS。BOSS战期间，武器会被强制锁定为 **剑 (Sword)** 以进行公平决斗。

### 通用规则（所有 BOSS 共享）
- **血量缩放**：BOSS 的基础血量会乘以 `bossHpMultiplier`（设置里可调，默认约为 2），因此同一个 BOSS 在不同设置下“看起来更肉”是正常现象。
- **阶段切换无敌帧**：每次进入新阶段会有短暂无敌（约 30 帧 ≈ 0.5 秒）并伴随闪烁提示，用于给玩家读条/走位窗口。
- **竞技场机制**：进入 BOSS 战会锁定视口并生成左右边界墙，防止“拉扯逃课”并保证战斗节奏。
- **村庄保护**：如果战斗发生在村庄安全区内，会对部分 BOSS 的攻击做保护性处理（例如飞行弹/冲刺等在村庄内不会对玩家造成伤害）。

### BOSS 门禁（Biome Gate）
门禁战是“切换群系前的关卡拦截”，用于把节奏从跑酷切到决斗：
- **触发时机**: 从一个群系切到下一个群系前，可能先进入门禁战；胜利后才会切换成功。
- **次数规则**: 默认同一群系“首次离开”最多触发 1 次门禁；Forest 默认免门禁。
- **BOSS 类型**: 某些群系会指定门禁 BOSS（例如 Nether 通常为 Ghast；Deep Dark 通常为 Wither Skeleton）；未指定时使用默认门禁 BOSS（Wither）。
- **凋零节奏控制**: 凋零门禁不会每次都触发，而是“每经过两次环境切换才触发一次”（村庄不计入环境切换）。

### ☠️ 凋零 (Wither)
*   **简介**: 拥有三个头颅的飞行亡灵生物。
*   **技能**: 发射追踪玩家的黑球/追踪弹，冲刺落地产生震荡波。
*   **阶段**: 拥有三个战斗阶段，随着血量降低，移动速度与攻击频率显著增强。
*   **基础数值**: 基础最大生命约 30（再乘以 BOSS 血量倍率）；触碰/技能会造成 1~2 格伤害。
*   **阶段阈值**: HP < 60% 进入阶段2；HP < 20% 进入阶段3。
*   **阶段1（压制）**:
    *   黑球：约每 180 帧发射 1 发（速度约 4，伤害 1）。
*   **阶段2（爆发）**:
    *   三连散射：约每 120 帧发射 3 发散射黑球（每发伤害 2）。
    *   冲刺：约每 480 帧进入一次冲刺，命中会造成较大击退；冲刺结束会在落点生成震荡波（持续扩散的范围伤害）。
*   **阶段3（濒死追杀）**:
    *   追踪弹：约每 60 帧会形成“追踪弹阵”，弹体会在延迟后开始追踪并持续推进，逼迫玩家频繁走位。

### 👻 恶魂 (Ghast)
*   **简介**: 漂浮在空中的巨大白色生物，常发出凄厉的叫声。
*   **技能**: 8字形机动飞行，发射可反射的火球，具备突进能力。
*   **机制**: 受伤积累到一定程度会触发“哭泣”状态，短暂停火，是输出的好时机。
*   **基础数值**: 基础最大生命约 25（再乘以 BOSS 血量倍率）；火球伤害约 2。
*   **阶段阈值**: HP < 50% 进入阶段2；HP < 20% 进入阶段3。
*   **火球**: 阶段越高越密集（约每 150/90/60 帧发射 1/2/3 发）；火球带 `reflectable` 标记，可被近战反弹。
*   **突进**: 会先短暂蓄力并提示⚠️，然后朝目标位置高速冲刺；命中会造成伤害并把玩家击退，冲刺结束后短暂眩晕进入破绽期。
*   **哭泣状态**: 累计受到约 10 次命中后会进入“哭泣”约 5 秒（此期间基本停火），并会出现泪水粒子。

### 🔥 烈焰人 (Blaze)
*   **简介**: 全身燃烧着火焰的浮空怪物。
*   **技能**: 三连火球射击，间歇性释放“火焰旋风”追踪火柱。
*   **阶段**: 进入阶段3后会召唤小烈焰人助战，并提升自身防御力。
*   **基础数值**: 基础最大生命约 28（再乘以 BOSS 血量倍率）。
*   **阶段阈值**: HP < 70% 进入阶段2；HP < 50% 进入阶段3。
*   **三连火球**: 始终激活；阶段越高间隔越短（约每 240/180/120 帧触发一次“三连”，三发之间间隔约 18 帧）。
*   **火焰旋风（阶段2+）**: 约每 600 帧生成 3 根追踪火柱，持续一段时间；火柱接触/灼烧会定期扣血。
*   **召唤小烈焰人（阶段3）**: 只会召唤一次 2 只小烈焰人；小怪存活期间 BOSS 会显著减伤（更鼓励先清小怪）。

### 🦴 凋零骷髅 (Wither Skeleton)
*   **简介**: 高大的黑色骷髅战士，手持石剑。
*   **技能**: 三段近战连招，起跳劈砸造成AOE伤害。
*   **机制**: 具备格挡能力，但格挡可被玩家连续攻击破防（造成眩晕）。
*   **基础数值**: 基础最大生命约 40（再乘以 BOSS 血量倍率）；属于“更耐打”的地面 BOSS。
*   **三段连招**: 横扫（近距扣血）→下劈（更高伤）→突刺（带击退）。
*   **跳劈**: 先蓄力提示⚠️，随后跳起下落砸地，落点附近会造成范围伤害并附带击退。
*   **格挡破防**: BOSS 进入格挡后会减少正面伤害；如果玩家连续命中堆叠到一定次数会触发“破防眩晕”，进入一段明显的输出窗口。

## 🏡 村庄系统

在冒险的间隙，玩家会遇到宁静的村庄。这里是安全的避风港（Safe Zone）。

- **生成机制**: 随分数区间刷出，含安全区，玩家进入后可休整。
- **安全区规则**: 村庄范围内不会自然刷怪，适合作为补给/学习/交易的休整点。
- **核心建筑**:
  - **睡觉屋 (Bed House)**:
    - 每个村庄仅可休息 1 次；休息会直接把生命值恢复到满血（若已满血则不会消耗休息次数）。
    - 休息完成会触发自动保存（降低“刚学会一个词就掉坑”的挫败感）。
  - **单词学习屋 (Word House)**:
    - 挑战每次 10 题；每答对 1 题直接奖励 **+1💎**（即时反馈）。
    - 结算额外奖励积分：完美通关 +100 分，否则 +50 分，并会按群系发放额外奖励（用于鼓励“在不同地图学习不同主题词”）。
    - 题库会优先使用当前群系的主题词（如海洋 fish/wave/boat…；沙漠 sand/cactus/oasis…），若群系未配置则回退到森林主题。
    - **题型与出题方式**:
      - 当前题型为“英文 → 选择中文含义”（界面会显示英文 `en`，选项来自 `zh`）。
      - 选项构成：正确答案 + 3 个干扰项；干扰项会从本局题目池的中文释义里抽取去重后打乱；如果释义不足会用“干扰项1/2/3”补齐。
      - 词条标准化：村庄词库既可写字符串也可写对象；会尝试从全局 `wordDatabase` 中补全 `zh/phrase/phraseTranslation`（用于朗读/提示信息更完整）。
      - 题目数量不足时允许重复：如果该群系的可用词条少于题目数，会重新洗牌循环取词，直到凑够题目数量。
    - **交互细节（更像小游戏）**:
      - 本挑战使用独立弹窗，不会打断通用学习挑战的 DOM；点击弹窗背景会直接退出本次挑战。
      - 每题有 1 次“提示”：随机禁用 2 个错误选项（本题仅一次），用于降低挫败感。
      - 点击任意选项后会锁定按钮并高亮正确/错误；同时触发朗读（若系统支持），并在屏幕上飘字提示“✅正确/❌错误”后进入下一题。
  - **商人屋 (Trader House)**:
    - **卖材料换钻石**：把铁/金/煤/贝壳/海星/火药/腐肉/线/末影珍珠/幽匿碎片/回响碎片等材料按单价换成💎。
    - **买盔甲**：用💎购买护甲并放入护甲背包（支持随时穿脱）。
    - **买防晒霜**：5💎 获得 7 分钟防护，能显著缓解沙漠高温，并可作为高温环境（地狱/火山）探索的防护手段之一；在下界还能把“岩浆即死”改为可承受的持续扣血（仍需谨慎）。
  - **特色建筑**: 每个群系的村庄都有独特的建筑风格，如沙漠的水井、雪地的温泉、海洋的灯塔等。
- **NPC**: 村民会主动迎客，迎客距离与速度可调；地图上最多同时保留 3 个村庄。

## 🛠️ 装备与武器

玩家拥有多种工具和武器，合理切换使用是生存的关键。

### 1. 武器库
*   **⚔️ 剑 (Sword)**:
    *   **用途**: 默认近战武器，攻速快，伤害适中。
    *   **适用**: 对付大多数地面敌人，BOSS战专用武器。
*   **🪓 斧 (Axe)**:
    *   **用途**: 强力近战，伤害高但攻速慢。
    *   **适用**: 破盾或对付高护甲敌人。
*   **⛏️ 镐 (Pickaxe)**:
    *   **用途**: 挖掘与破坏。
    *   **功能**: 可以破坏平台方块，有机会挖掘出藏宝块生成 **宝箱 (Chest)**。
    *   **细节**: 支持“按方块切割平台”——挖掉 1 格后平台会自动分裂成左右两段；同时会触发重力检查，可能导致上方悬空结构坍塌，需要规划落脚点再动手。
*   **🏹 弓 (Bow)**:
    *   **用途**: 远程攻击。
    *   **功能**: 长按蓄力可以增加射程和伤害。
    *   **消耗**: 需要消耗箭矢 (Arrow)。
    *   **适用**: 攻击远处的敌人或空中的幻翼、恶魂。

### 2. 护甲与道具
*   **护甲**: 提供不同防御与稀有度，包括皮革、链甲、铁、金、钻石、下界合金。
    *   护甲提供固定防御值（皮革1 / 链甲2 / 铁3 / 金2 / 钻石4 / 下界合金5），并带有耐久百分比。
    *   耐久会随“穿戴时间”持续消耗，到 0 会自动失效；不同护甲的持续时间不同（例如皮革更短，钻石更长），因此更适合把护甲当作“限时强化”来使用与轮换。
*   **静音靴**: 深暗之域探索必备，减少跳跃/攻击/用道具带来的噪音增量。
    *   制作方式：背包里使用 **幽匿碎片 x5** 可直接制作并自动装备（初始耐久 30）。
    *   噪音减免：会把关键动作的噪音增量压低到约 55%，且每次动作消耗 1 点耐久，耐久耗尽会损坏。
*   **功能道具**:
    *   **钻石**: 回血货币。
    *   **召唤物**: 南瓜/铁块可召唤傀儡助战。
    *   **其他**: 火药炸弹、末影珍珠传送、贝壳护盾、花朵持续回血等。

### 3. 攻击与互动方式
*   **近战**: 正面短程打击，注意部分 BOSS (如凋零骷髅) 有格挡与反击机制。
*   **远程**: 弓箭蓄力，伤害依赖蓄力时间；注意恶魂的火球可被近战攻击反射。
*   **切换**: 非 BOSS 战可自由切换武器；飞行 BOSS 会自动补给基础弓箭（弓=1、箭≥12），但 BOSS 战期间仍锁定剑，补给主要用于战后切回弓或非锁定状态的远程战斗。

## 🎮 安装说明

### 直接安装 APK
1. 从 Release 或构建产物下载 APK
2. 将 APK 传到手机并安装（如提示未知来源请允许）

### 本地打包安装
1. 按“本地构建流程”生成 APK
2. 在 `android-app/android/app/build/outputs/apk/` 获取 APK
3. 复制到设备并安装

## 📂 目录结构（仓库根：apk/）
- `Game.html`：游戏入口页面
- `src/`：核心逻辑与样式
- `config/`：关卡、词库与控制配置
- `scripts/sync-web.js`：生成离线单文件并同步到 `android-app/web/index.html`
- `android-app/`：Capacitor 与 Android 工程
- `.github/workflows/android.yml`：CI 构建与发布

## 💻 环境要求
- Node.js 22+
- JDK 21（CI 使用 Temurin 21）
- Android SDK（本地构建需要，CI 会自动安装）

## 🔨 本地构建流程
1. **安装依赖**
   - `cd android-app && npm ci`
2. **生成离线单文件并同步到 Android Web 目录**
   - `cd .. && node scripts/sync-web.js`
3. **同步到 Android 工程**
   - `cd android-app && npx cap sync android`
4. **构建 APK**
   - 在 `android-app/android/local.properties` 配置 `sdk.dir=...`
   - `cd android-app/android && ./gradlew assembleDebug`

## 📱 设备模式与触控
- 默认 `deviceMode` 为 `auto`
- 触控按键布局与 UI 缩放在 `src/styles.css` 与 `Game.html` 中配置
- 设置面板可切换手机/平板与横竖屏显示模式

## 🚀 GitHub Actions（在线构建）
- **workflow**：`.github/workflows/android.yml`
- **构建产物**：
  - Debug：`android-app/android/app/build/outputs/apk/debug/*.apk`
  - Release：`android-app/android/app/build/outputs/apk/release/*.apk`
- **发布位置**：`apk-latest` Release（同名 tag 更新、资产替换）

## 🔐 Release 签名（可选）
在仓库 Settings → Secrets 中添加：
- `ANDROID_KEYSTORE_BASE64`
- `ANDROID_KEYSTORE_PASSWORD`
- `ANDROID_KEY_ALIAS`
- `ANDROID_KEY_PASSWORD`

配置齐全后会发布可覆盖安装的 release APK；未配置时发布 debug APK。
